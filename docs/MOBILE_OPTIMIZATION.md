## Using Key Features on Mobile

### Dashboard

**Mobile View**:
- Statistics cards stack vertically
- Large, easy-to-read numbers
- Quick access to main sections
- Summary information at a glance

**Tips**:
- Scroll down to see all statistics
- Tap any card to navigate to that section
- Swipe to see charts and graphs

### Members Management

**Viewing Members**:
1. Tap hamburger menu (☰)
2. Select "Members"
3. Use search bar at top
4. Scroll table horizontally to see all columns
5. Tap member name to view details

**Adding Members**:
1. Tap "Add New Member" button
2. Fill form (fields stack vertically)
3. Scroll down to complete all fields
4. Tap "Save Member" button at bottom

**Editing Members**:
1. Find member in list
2. Swipe table to see "Actions" column
3. Tap "Edit" button
4. Update information
5. Tap "Update Member"

### Recording Contributions

**Mobile Workflow**:
1. Navigate to Contributions
2. Tap "Record Contribution"
3. Select member from dropdown
4. Enter amount (numeric keyboard appears)
5. Select date (native date picker)
6. Choose payment method
7. Scroll and tap "Record Contribution"

**Tips**:
- Use search in member dropdown
- Date picker is touch-friendly
- Amount field shows proper keyboard
- Transaction reference is optional

### Managing Loans

**Viewing Loans**:
- Loans list shows key information
- Swipe table to see all details
- Tap loan to view full details
- Action buttons are easy to tap

**Applying for Loans**:
1. Navigate to Loans
2. Tap "Apply for Loan"
3. Fill application form
4. Upload collateral photos (if needed)
5. Submit application

**Making Loan Payments**:
1. Find loan in list
2. Tap "Record Payment"
3. Enter payment amount
4. Select payment method
5. Submit payment

### Welfare Requests

**Submitting Requests**:
1. Navigate to Welfare
2. Tap "Submit Request"
3. Select request type
4. Enter amount
5. Provide reason/description
6. Attach supporting documents
7. Submit for approval

### Meetings

**Viewing Meetings**:
- See upcoming and past meetings
- Tap meeting to view details
- View attendance list
- Access meeting minutes

**Recording Attendance**:
1. Open meeting details
2. Tap "Record Attendance"
3. Check/uncheck member names
4. Tap "Save Attendance"

## Mobile Tips and Tricks

### General Usage

1. **Rotate Your Screen**
   - Landscape mode gives more space for tables
   - Portrait mode better for forms and reading

2. **Use Pull-to-Refresh**
   - Pull down on most pages to refresh data
   - Ensures you have latest information

3. **Zoom if Needed**
   - Pinch to zoom on reports and charts
   - Double-tap to zoom on images

4. **Search Functionality**
   - Use search boxes to filter long lists
   - Faster than scrolling through tables

5. **Save to Home Screen**
   - Add web app to home screen for quick access
   - Works like a native app

### Adding to Home Screen

#### iPhone/iPad (Safari)
1. Open the app in Safari
2. Tap the Share button (square with arrow)
3. Scroll and tap "Add to Home Screen"
4. Name it "OT Savings" or similar
5. Tap "Add"

#### Android (Chrome)
1. Open the app in Chrome
2. Tap three dots (⋮) menu
3. Tap "Add to Home screen"
4. Name it "OT Savings"
5. Tap "Add"

### Form Entry Tips

1. **Use Autofill**
   - Enable autofill for faster form completion
   - Save common entries in your phone

2. **Voice Input**
   - Use voice-to-text for long descriptions
   - Tap microphone icon on keyboard

3. **Copy-Paste**
   - Long-press to copy text
   - Paste transaction references, etc.

### Network Issues

**If Page Won't Load**:
1. Check your internet connection
2. Pull down to refresh
3. Close and reopen the app
4. Clear browser cache if needed

**Saving Data**:
- App uses minimal data
- Images may use more data
- Use WiFi for uploads when possible

## Troubleshooting Mobile Issues

### Issue: Can't See Navigation Menu

**Solution**:
1. Look for hamburger icon (☰) in top right
2. Tap to open menu
3. If still not visible, reload page
4. Clear browser cache

### Issue: Tables Cut Off

**Solution**:
1. Swipe table horizontally
2. Rotate to landscape mode
3. Pinch to zoom out
4. Some columns hidden on purpose (less important data)

### Issue: Buttons Too Small

**Solution**:
1. Update to latest app version
2. Zoom in on the page
3. Use larger screen device if available
4. Try landscape orientation

### Issue: Form Inputs Zooming

**Solution**:
- This is fixed in latest version
- If still happening:
  1. Clear browser cache
  2. Reload the page
  3. Update browser to latest version

### Issue: Keyboard Covering Input

**Solution**:
1. Scroll down before typing
2. Keyboard should auto-adjust
3. Tap outside to dismiss keyboard
4. Try landscape mode

### Issue: Slow Performance

**Solution**:
1. Close other browser tabs
2. Restart browser
3. Clear browser cache
4. Check internet speed
5. Use WiFi instead of mobile data

### Issue: Login Not Working on Mobile

**See**: [MOBILE_LOGIN_TROUBLESHOOTING.md](MOBILE_LOGIN_TROUBLESHOOTING.md)

## Mobile Performance Tips

### Speed Up Loading

1. **Clear Cache Regularly**
   - Once per week
   - After major updates

2. **Close Unused Tabs**
   - Frees up memory
   - Improves performance

3. **Use WiFi**
   - Faster than mobile data
   - More reliable connection

4. **Update Browser**
   - Latest version performs better
   - Security improvements

### Save Data

1. **Limit Image Uploads**
   - Compress before uploading
   - Use WiFi for uploads

2. **Use Search**
   - Instead of loading all records
   - Faster and uses less data

3. **Log Out When Done**
   - Saves battery
   - Protects your account

## Security on Mobile

### Protect Your Account

1. **Use Strong Password**
   - Change default password
   - Don't share with others

2. **Log Out After Use**
   - Especially on shared devices
   - Prevents unauthorized access

3. **Don't Save Password**
   - On shared devices
   - Use password manager instead

4. **Lock Your Phone**
   - Use PIN/fingerprint/face ID
   - Protects if phone is lost

### Safe Practices

1. **Use Secure Networks**
   - Avoid public WiFi for sensitive tasks
   - Use mobile data if WiFi is unsecured

2. **Keep Browser Updated**
   - Security patches
   - Bug fixes

3. **Report Issues**
   - Contact admin if something seems wrong
   - Report unauthorized access
